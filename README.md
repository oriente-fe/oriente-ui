# Oriente UI

> A bunch of components for supporting Oriente front-end team.

[DEMO](https://oriente-ui.github.io/)

## Setup

```bash
$ npm install oriente-ui -S
```
## Usage

```js
import orienteUI from 'oriente-ui'
Vue.use(orienteUI)
```

## Build

Publish npm package

```bash
$ yarn lib
```

## Development

Start styleguidist

```bash
$ yarn styleguide
```

Start demo site

```bash
$ yarn serve
```
