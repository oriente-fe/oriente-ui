# Oriente UI

> A bunch of components for supporting Oriente front-end team.

[DEMO](https://workspace.tonypai.now.sh)

## Features

- Mobile friendly.
- Works on all fancy browsers.
- Supports SSR, SPA, PWA and standard HTML pages.

## Setup

```sh
$ yarn add oriente-ui
```
## Usage

```js
import orienteUI from 'oriente-ui'
Vue.use(orienteUI)
```

## Build

Publish npm package

```bash
$ yarn lib
```

## Development

Start styleguidist

```bash
$ yarn styleguide
```

Start demo site

```bash
$ yarn serve
```
